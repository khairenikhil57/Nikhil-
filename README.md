# HR INDUSTRIES — Static Website

This is a single-page static website for HR INDUSTRIES. Ready to host on GitHub Pages.

## Files
- `index.html` — main page
- `styles.css` — styles
- `script.js` — small JS
- `assets/` — put images (logo, product photos) here

## Local preview
Open `index.html` in a browser.

## Deploy to GitHub Pages (quick steps)

1. Create a new repository on GitHub (e.g. `hr-industries-website`).
2. Initialize local repo and push:
   ```bash
   git init
   git add .
   git commit -m "Initial website"
   git branch -M main
   git remote add origin https://github.com/<your-username>/hr-industries-website.git
   git push -u origin main
